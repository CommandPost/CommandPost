Titles To Keywords

When you drag a Final Cut Pro Event to this Toolbox (which must only contain a single Project), we will look through all the Titles connected

to the clips in the Primary Storyline, read the name of those clips (as they appear in the Timeline Index), then create new

Keyword ranges for each of those Titles in the same Event, and send that data back into Final Cut Pro.


This utility was commissioned by Knut Hake.
